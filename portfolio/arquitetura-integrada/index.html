Arquitetura Integrada

Website e Front-End

Arquitetura Integrada é um escritório localizado na cidade de Natal-RN cujo objetivo é gerar melhorias para negócios por meio da arquitetura.

Objetivo

O objetivo do projeto era criar um layout de website para um escritório que trabalha com projetos comerciais. Assim, o layout website precisava conversar com um público formado por empreendedores, que transmitisse confiança no conhecimento e na exeprtise das arquitetas na transformação positiva de negócios.

Processo de design

  • _Pesquisa
  • _Análise de semelhantes
  • _Geração de ideias/brainstorming
  • _Entrevista com stakeholders
  • _Moodboard
  • _Definição de sitemap
  • _Wireframes
  • _Visual Guide (design system)
  • _Layout visual
  • _Implementação

Solução

A estratégia criada envolveu criar um layout clean, obedecendo as demarcações do manual de marca, com visual business fazendo com que os usuários conhecessem o trabalho das arquitetas e tivessem um ponto de contato.

Interface

Para a construção do layout do site o processo passou pelas etapas de estruturação das páginas com a criação de um sitemap e fluxo de navegação, criação de um Visual Guide (Design System) para manter a consistência do projeto, wireframe e layout final.
